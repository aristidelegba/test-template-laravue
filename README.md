# Vuejs developer test

## Setup

Make sure you have NodeJS and PHP installed

```bash
# install vuejs app dependencies and launch vuejs app
$ yarn install
$ yarn dev

#then bootstrap laravel app
$ composer install
$ php artisan key:generate
$ php artisan serve

```
